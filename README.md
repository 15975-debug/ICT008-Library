## Student Project
This project was developed for ICT0008 Advanced Programming Skills.

## Project Title
Library Book Management System

## Description
This C++ application was created for a local community library. The system uses a Book class to store and manage book details such as title, author, ISBN, availability, and date added. The program allows users to borrow and return books and checks whether a book is available before borrowing.

## Features
- Store book details
- Display book information
- Borrow a book using ISBN
- Return a book
- Validate availability
- Manage an array of 5 books
